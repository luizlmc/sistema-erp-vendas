# AGENTS - Empresas

## Escopo
Cadastro de empresas emissoras e configuracoes fiscais.

## Objetivo
Garantir manutencao completa de dados da empresa para emissao fiscal.

## Dependencias e Fontes
- `listCompaniesRequest`, `createCompanyRequest`, `updateCompanyRequest`, `deleteCompanyRequest`

## Regras de Implementacao
- Modal com visualizacao/edicao e snapshot para cancelar.
- Confirmacao para exclusao/inativacao.
- Layout e botoes no padrao global.

## Checklist
- Abrir item da lista abre detalhe.
- Edicao persiste no backend.
- Espacamento uniforme entre secoes.
